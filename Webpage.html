Movie Rating Prediction with SVD, Neural CF, and Bayesian PMF

DS4420 extra credit app prototype with a landing page and an interactive Bayesian visualization

Project Goal

This project studies explicit movie rating prediction in recommender systems. Using the same MovieLens task and the same shared train validation test split, we compare three approaches that represent classical, neural, and probabilistic modeling.
Dataset: MovieLens
Inputs: user_id and movie_id
Metric: RMSE
Split: 80 10 10

Overall Comparison

Model Validation RMSE Test RMSE
SVD 0.8648 0.8612
Bayesian PMF 0.8704 0.8654
Neural CF 0.8988 0.8922
Under the shared setting, SVD achieved the best predictive performance. Bayesian PMF followed closely and added uncertainty aware outputs.

SVD Based Matrix Factorization

0.8612
A strong classical baseline using global mean, bias terms, and latent factor interactions learned with stochastic gradient descent.

Neural Collaborative Filtering

0.8922
A neural recommendation model that learns user and movie embeddings and models nonlinear interactions through a multilayer perceptron.

Bayesian PMF

0.8654
A probabilistic matrix factorization model that provides competitive accuracy together with interpretable uncertainty aware prediction summaries.

Why This Comparison Matters

These three models reflect different assumptions about how user item interactions should be learned. SVD emphasizes compact latent structure, Neural CF emphasizes nonlinear interaction learning, and Bayesian PMF adds probabilistic interpretation. Comparing them on the same task makes the differences easier to interpret.
